# Project 3 — Vite + React

A small, interactive PokéAPI viewer and stat summarizer with styling based on Pokémon type

## Prerequisites

- Node.js
- npm

## Setup

Open a terminal in the project root and run:

```powershell
npm install
```

Start the development server:

```powershell
npm run dev
```
